- VERIFICAR QUE detalle_cita_id COINCIDE CON SU HASH
- VERIFICAR SI YA FUE CONFIRMADO
- ACTUALIZAR EL REGISTRO EN detalles_citas CORRESPONDIENTE, CON LA RESPUESTA(SI, NO) Y 
  FECHA Y HORA.

#EDITAR UNA REUNION
- VALIDAR DATOS
- GUARDAR DATOS
- ENVIAR CORREO DE REPROGRAMACION A LOS ASISTENTES ACTUALES
- ENVIAR CORREO DE INVITACION A LOS NUEVOS ASISTENTES
- ENVIAR CORREO DE ELIMINACION A LOS ASISTENTES ELIMINADOS

function sendEmail($cita, $asistentes, $tipo)
{
  foreach ($asistentes as $asistente) {
    Mail::to($asistente->email)->send(new CitaEmail($cita, $asistente, $tipo));
  }
}
